GenZ: Internal Data Integration and Monitoring System for Telkom Regional 5

Laravel MySQL REST API

The GenZ website is an internal platform developed for Telkom Regional 5 to streamline operations and centralize company data. It features modules such as Activity Logs, Inventory Management with report generation (PDF/Excel), Document Handling, Guest Book, and Room Booking. Equipped with an interactive dashboard for activity monitoring and structured digital documentation, GenZ provides an efficient and integrated solution to support daily workflows and improve data management across the organization.

Fridge Recipe: Smart Recipe Finder Based on Available Ingredients

Node.js RESTful API Google Cloud Platform JavaScript

Fridge Recipe is a web-based application that helps users discover cooking recipes based on the ingredients they already have in their fridge. Users can input available ingredients, search for matching recipes, and save favorites by logging in. The platform is powered by a REST API and deployed on Google Cloud Platform (GCP), ensuring scalability, reliability, and seamless access to diverse recipe recommendations. This application aims to reduce food waste while making cooking easier and more enjoyable.

Bupati Decree Submission System

Laravel Bootstrap MySQL

A web-based application for submitting and monitoring Bupati Decree (SK) requests online. The system allows users to apply, track the process status, and download approved decrees quickly, transparently, and efficiently. It provides an intuitive interface with clear steps for submission, review, and approval. Users can also verify their application status anytime using a unique submission code, ensuring accessibility and accountability. This solution helps improve efficiency in public administration by reducing manual paperwork and saving time.

Smart Admission using SAW

Laravel Bootstrap MySQL

This web-based system is developed to support the student admission process at SKB Ujung Pandang using the Simple Additive Weighting (SAW) method. It enables participants to register online, while administrators can manage data, set criteria, and run the SAW calculation to determine final scores and admission status. With automated ranking, clear result visualization, and an integrated dashboard, the system ensures efficiency, transparency, and fairness throughout the selection process.
